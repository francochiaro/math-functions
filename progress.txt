## Codebase Patterns
- Frontend uses Next.js 16 with App Router, TypeScript, and Tailwind CSS
- D3.js for all chart rendering and interactions
- Backend uses Python FastAPI for curve fitting API
- Geist font family for typography (sans and mono variants)
- Color scheme: zinc neutrals, green (#22c55e) for success/accents, indigo (#6366f1) for data points
- All state management done with React hooks (useState, useCallback, useRef)
